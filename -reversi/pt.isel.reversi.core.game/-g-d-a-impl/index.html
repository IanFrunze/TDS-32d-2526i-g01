interface GDAImpl

Contract for persistence operations related to a Reversi game.

Implementations of this interface are responsible for storing and loading the textual representation of a game's state (headers, piece history and pass lines). Implementations may use the local filesystem, a database, or a remote service.

Note: IO failures from the underlying storage are surfaced as standard IOExceptions from implementation code; callers should treat those as transient I/O errors.
