Game

open class Game(val dataAccess: GDAImpl, val players: List<Player>, val target: Boolean, val playerTurn: PieceType = firstPlayerTurn, val currGameName: String?, val board: Board?) : GameImpl

Lightweight test/dummy implementation of GameImpl used for data access and integration tests.

Only acts as a structural carrier for required properties; behavioural methods are left as TODOs so they surface if accidentally invoked in logic outside targeted tests. Use the nested helper subclasses to build simple game states for tests (empty, one player or two players).

Note: This class is intentionally minimal and not suitable for exercising game logic.
