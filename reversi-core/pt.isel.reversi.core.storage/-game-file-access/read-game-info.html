readGameInfo

fun readGameInfo(file: File): Pair<List<PieceType>, Int>

Reads game header information (available pieces and side) from the provided file. The function scans the file for header lines and returns the first occurrences.

Return

Pair(list of available pieces, board side)

Parameters

file

the file to scan

Throws

InvalidPieceInFileException

if a piece line is malformed while scanning

InvalidSideInFileException

if the side line is missing or invalid

InvalidAvailablePiecesInFileException

if the available pieces line is missing
