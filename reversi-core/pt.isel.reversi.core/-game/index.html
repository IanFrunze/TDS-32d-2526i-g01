Game

data class Game(val target: Boolean = false, val currGameName: String? = null, val lastModified: Long? = null, val gameState: GameState? = null, val countPass: Int = 0, val myPiece: PieceType? = null, val config: CoreConfig = loadCoreConfig())

Represents a Reversi game, managing the game state, player turns, and interactions with storage. The game has two modes: local and not local.

Local Game

In a local game, both players are managed within the same game instance. No storage operations are performed.

Not Local Game

In a not local game, only one player is managed within the game instance. The game state is saved and loaded
