GameFileAccess

object GameFileAccess

Low-level helpers to read/write the textual persistence format used by LocalGDA.

The format is line-oriented and intended to be human-readable. This object exposes small, well-documented functions that handle serialization and parsing of the different line kinds: available pieces, side, piece placements and pass turns.
